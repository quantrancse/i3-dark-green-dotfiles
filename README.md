## Some details
+ **OS**: Manjaro i3 minimal
+ **WM**: i3 gaps
+ **Status bar**: polybar
+ **Terminal**: termite
+ **Shell**: zsh
+ **File Manager**: Nemo, ranger
+ **Launcher**: rofi
+ **Editors**: vim
+ **Browser**: Brave
+ **Music**: mpd, ncmpcpp, Spotify
+ **Theme**: [Flat-Remix-GTK-Green-Dark](https://www.gnome-look.org/p/1297346/)
